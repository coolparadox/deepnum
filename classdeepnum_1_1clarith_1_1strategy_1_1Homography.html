Detailed Description

First degree homographic transformation.

This strategy accepts a Number \(x\) as input and outputs \(y=\frac{n_1 x + n_0}{d_1 x + d_0}\) where \(n_1\), \(n_0\), \(d_1\) and \(d_0\) are machine native signed integers.
